Откройте терминал в этой папке и введите 
npm install 
после установки пакетов введите
npm start
Должен открыться  браузер с web приложением по адресу 
http://localhost:3000/

1) Задача - Показать конечный вид таблицы с добавленными типами билетов. Объяснить свое решение.

Объяснение:
в папке src/data/ находится файл typesOfTickets.js в нём находится объект поля которого являются типами билетов. Перед выводом строки в базу данных создаётся экземпляр класса BDRow, который при создании пробегается по массиву всех типов файлов и создаёт соответствующие поля объекта.
Допустим , если указать в typesOfTickets.js новое поле newType:"newType", то новая строка будет иметь два дополнительных поля : ticket_newType_price и ticket_newType_quantity .
Таким образом можно добавлять сколько угодно большое количество типов билетов.
 
2) Задача - Показать конечный вид таблицы с добавленными типами билетов. Объяснить свое решение.

Объяснение:
При загрузки приложения можно выбрать любое событие на выбор.
При выборе события - появляется панель управления со счётчиками билетов(для каждого типа свой счётчик).
При нажатии на кнопку с символом "+" будет создаваться одна строка с единственным билетом нужного типа, для этой строки будет свой уникальный баркод.
